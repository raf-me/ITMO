## Лабораторная работа №1

### Представление данных в памяти компьютера

Цель работы: Изучить представление различных типов данных в памяти компьютера, побайтовую структуру чисел, порядок хранения байтов (Little Endian / Big Endian), а также научиться работать с памятью на языках C, Assembly и Python.

⸻

### Теоретические сведения

В памяти компьютера все данные хранятся в виде последовательности байтов.
Различные типы данных занимают разное количество байтов:

| Тип         |  Размер   |
| ----------- | --------- |
| char        |  1 байт   |
| short       |  2 байта  |
| int         |  4 байта  |
| long        |  8 байт   |
| float       |  4 байта  |
| double      |  8 байт   |


Существует два способа хранения байтов числа:
	•	Little Endian — младший байт по меньшему адресу
	•	Big Endian — старший байт по меньшему адресу

Современные процессоры x86 используют Little Endian.

### Задание

#### Часть 1 — C
Написать программу, которая:

1. Выводит размеры типов данных
2. Создает переменные разных типов
3. Выводит адреса переменных
4. Выводит побайтовое представление числа
5. Определяет порядок байтов в системе

Программа должна:
1. Использовать указатели
2. Выводить каждый байт числа
3. Вывести адрес каждого байта

#### Часть 2 — Assembly
Написать программу на Assembly, которая:
1. Записывает число в память
2. Считывает байты по одному
3. Выводит их на экран
4. Показывает адреса

#### Часть 3 — Python
Написать программу, которая:
1. Преобразует числа в байты
2. Показывает представление int и float
3. Использует struct.pack
4. Показывает порядок байтов


#### Требования к отчету

В отчете должно быть:
- Цель работы
- Краткая теория
- Листинг программ (C, Assembly, Python)
- Скриншоты выполнения
- Таблица размеров типов
- Вывод о порядке байтов
- Вывод по работе
